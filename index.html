Project Title 1: Dynamic Functional Connectivity Analysis using Topological and Graph Theoretical Methods

Extracted brain state networks using topological and graph models. Applied graph theory and topological data analysis metrics to study the structure of temporal brain state organization. Studied graph spectral features and statistical measures of the dynamic connectivity of schizophrenia and bipolar disorder subjects.
